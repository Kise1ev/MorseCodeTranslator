<h1>📝 Description</h1>
<h3>This program translates Morse code in Latin into text. The program prompts the user to enter Morse code through the console. The Morse code is then broken down into individual words and each word is broken down into individual letters. Each letter of the Morse code is checked against the morseTranslations hash table to find the corresponding Latin character. If a character is found, it is added to the string of the translated Latin word. After all the letters of the word have been processed, the translated word is added to the latinWords vector. At the end, the program outputs the translated Latin message to the console.</h3>
